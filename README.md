# Smart Lead Generator - DSAI Lab Project

This is a group project for the IIT Madras DSAI Lab. This project aims to build Full stack Hyper Personalized Lead Generator.

## Project Structure

This repository is organized into milestones. All relevant code, documentation, and data for each project phase are contained within their respective folders.

-   **milestone 1**: Problem definition and literature review.
-   **milestone 2**: Dataset preparation
-   **milestone 3**: Model architecture
-   **milestone 4**: Model Training
-   **milestone 5**: Model Evaluation
-   **milestone 6**: Model Deployment

Pretrained models deployed on hugging face.

- [**Personality classification model**](https://huggingface.co/spaces/PavanB99Indian/mbti-classifier) 
- [**Email generation model**](https://huggingface.co/anishark/LEAD_EMAIL_GENERATION_MODEL)
