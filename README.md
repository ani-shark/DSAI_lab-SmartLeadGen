# Smart Lead Generator - DSAI Lab Project

This is a group project for the IIT Madras DSAI Lab. This project aims to build a smart lead generator.

## Project Structure

This repository is organized into milestones. All relevant code, documentation, and data for each project phase are contained within their respective folders.

-   **[milestone 1/](milestone%201/)**: Problem definition and literature review.
-   **[milestone 2/](milestone%202/)**: Dataset preparation
-   **[milestone 3/](milestone%203/)**: Model architecture
-   **[milestone 4/](milestone%204/)**: Model Training
